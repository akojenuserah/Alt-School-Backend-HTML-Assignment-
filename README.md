# Alt-School-Backend-HTML-Assignment-
## Overview
This project was created to fulfill the requirements of the first assignment of my ALT School Backend Engineering Diploma program.

## Date

28th April, 2025

## Instructions

- Create a recipe webpage using HTML to demonstrate your understanding of basic HTML structure, organization, and formatting. You will build a page for a recipe of your choice.
- The content should include the following:
  - Recipe name and short description
  - Recipe details (prep time, cook time, servings, difficulty)
  - Ingredients list (unordered list with at least 6 items)
  - Instructions (ordered list with at least 5 steps)
  - Optional Tips or Notes section

- Adding extras such as:
  - A link to the recipe source
  - A “Nutrition Facts” section
  - Semantic elements like <section> or <article>
